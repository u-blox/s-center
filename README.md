# s-center
The s‑center Wi‑Fi, Bluetooth and Bluetooth low energy module and configuration software is a powerful and easy‑to‑use tool for evaluating, configuring, and testing u‑blox standalone short range modules. It includes an intuitive, easy‑to‑understand and easy‑to‑use graphical interface.

**s-center** is a comprehensive Windows application for testing, configuring, and exploring u-blox short-range standalone modules.

## Features

### Serial Communication
- Direct AT Command Interface with syntax highlighting
- Full-featured terminal emulator
- Command history and AT command browser

### WiFi Management
- WiFi Station Mode: Scan, configure, connect with WPA2/WPA3
- WiFi Access Point Mode: Configure AP, monitor clients

### Bluetooth Features
- Bluetooth Configuration: Device settings, advertising, BLE
- Connection Management: Scan, ACL/SPP connections
- GATT Server/Client: Configure services and characteristics

### Network Capabilities
- Socket Connections: TCP/UDP client/server
- MQTT Client: Broker connections, pub/sub, TLS/SSL
- HTTP/HTTPS: REST API testing

### Security
- Certificate Management: Import CA, client certs, private keys
- TLS/SSL support for all connections

### Advanced Features
- Firmware Updates via XMODEM
- User-Defined Commands
- Debug Logging
- Error Code Database

## Supported Modules
- **NORA-W36** - WiFi 4 & Bluetooth 5.3 module
- **NORA-B26** - Bluetooth 6.0 module
- **NORA-B27** - Bluetooth 6.0 module - Peripheral role only

> **Note:** For other u-blox modules (ANNA-B, NINA-B/W, ODIN-W series), please use s-center classic.

## Requirements
- Windows 10 (1809+) or Windows 11
- .NET 9 Runtime
- Serial port connection
- **64-bit Windows** (x64 only)
